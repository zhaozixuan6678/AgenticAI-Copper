# 铜合金性能预测智能代理系统

## 系统概述

这是一个基于LangChain框架的智能代理系统，专门用于铜合金性能预测。系统将原始代码的核心功能封装为多个独立的LangChain工具，具备自然语言理解能力，能够接收用户的自然语言指令，自动选择并调用合适的工具。

## 系统架构

```
UI/
├── __init__.py          # 包初始化
├── agent_core.py        # 核心代理类，工具注册和管理
├── intent_recognition.py # 意图识别模块，解析用户指令
├── tool_selector.py     # 工具选择器，选择和执行工具
├── ui_interface.py      # 用户界面，提供交互体验
├── config.py            # 系统配置文件
├── main.py              # 主程序入口
└── README.md            # 本文件
```

## 核心功能模块

### 1. 特征工程工具 (`feature_engineering`)
- **功能描述**：将原始合金数据转换为机器学习特征
- **输入参数**：原始Excel文件路径、输出Excel文件路径、sheet名称
- **输出格式**：包含18个特征列的Excel文件
- **支持的特征**：CR_x_ATem、ATem_x_Atime、STem-ATem、One-Hot-Processing、HHI、N_eff、Tmavg、Family、Si/wt.%、δ、VECavg、Mg/(Ni+Si)、χvar、Smix、Hardness/HV、EC/%IACS、Q3-Euclidean

### 2. 性能预测工具 (`performance_prediction`)
- **功能描述**：预测铜合金的三种关键性能指标
- **输入参数**：合金成分字典、工艺参数字典、工艺路线字符串
- **输出格式**：JSON格式的预测结果
- **预测指标**：
  - `Hardness/HV`：维氏硬度（HV）
  - `EC/%IACS`：电导率（国际退火铜标准百分比）
  - `Q3-Euclidean`：综合性能评价指标

### 3. 模型训练工具 (`model_training`)
- **功能描述**：训练和优化铜合金性能预测模型
- **输入参数**：训练数据路径、待预测数据路径、参数配置路径、输出路径
- **输出格式**：训练好的模型和评估报告
- **支持的模型**：SVR（支持向量回归）、CAT（CatBoost回归器）

## 智能代理特性

### 意图识别能力
- 支持多种自然语言表达方式
- 自动识别用户意图（特征工程、性能预测、模型训练）
- 基于正则表达式的模式匹配

### 工具选择逻辑
- 根据意图自动选择最优工具
- 参数自动提取和验证
- 错误处理和友好的错误提示

### 结果整合输出
- 格式化的结果展示
- 包含意图识别、工具使用、参数和结果的完整信息
- 支持多种输出格式（文本、JSON等）

## 系统特点

✅ **自然语言理解**：支持中文自然语言指令
